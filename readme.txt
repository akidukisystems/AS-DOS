１．はじめに

    このプログラムは、アセンブリ言語の練習を目的として作成されたプログラムです。
    改変、再配布等自由に行うことができます。詳しくは後述の「２．著作権」をご覧ください。

２．著作権

    このプログラムは、AkidukiSystemsが開発した AS-DOS を使用したことを示す表示をする限り、
    改変、再配布や名称の変更を自由にできます。これは AS-DOS を使用する限り、永続的に継承されます。
    ただし、他のプログラムのライセンスや著作権には干渉しません。

３．システム要件

    このプログラムを実行する上で、以下のプログラムやハードウェアを要求します。

    [共通]
    ・最新安定版の nasm、mtools、make（アセンブルやフロッピーイメージファイルの作成に必要）
    　及びそれらを実行するための 最新安定版の Ubuntu
    　（または同様の機能を持つ Linux ディストリビューション）
    
    [エミュレータ上での動作の場合]
    ・最新版の qemu（qemu-system-i386）

    [実機での動作の場合]
    ・フロッピーディスクドライブ 1機
    ・Intel 80386 以上のCPU、または リアルモードを搭載した 16bit長 命令コードを実行可能なCPU
    ・64KiB以上のメモリ

４．備考

    ・半角スペース文字ではなく、タブ文字をインデントに使用しているので、
    　一部の環境では正常にインデントされない場合があります。
    　タブ文字１文字につき半角スペース４文字分のスペースを占有すること前提でコードを記述していますので、
    　正常にインデントされない場合はインデントの設定を変更するなどしてください。
    　（Githubでは、URLの後ろに " ?ts=4 " と入力することで正常にインデントされます。
